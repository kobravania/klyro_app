# Настройка кнопки Menu Button для Klyro Bot

## Что такое Menu Button?

Menu Button — это кнопка внизу чата с ботом, которая открывает ваше Web App. У Crypto Bot она называется "ОТКРЫТЬ", у вас может быть любое название.

## Как настроить через BotFather:

### Способ 1: Через команды BotFather

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/mybots`
3. Выберите вашего бота **Klyro Bot**
4. Выберите **Bot Settings**
5. Выберите **Menu Button**
6. Выберите **Edit Menu Button**
7. Выберите **Web App**
8. Введите текст кнопки (например: `Открыть Klyro` или `ОТКРЫТЬ`)
9. Выберите ваше Web App из списка (то, которое вы только что создали)

### Способ 2: Через Bot API (программно)

Если у вас есть токен бота, можно настроить через API:

```bash
curl -X POST "https://api.telegram.org/bot<ВАШ_ТОКЕН>/setChatMenuButton" \
  -H "Content-Type: application/json" \
  -d '{
    "menu_button": {
      "type": "web_app",
      "text": "ОТКРЫТЬ",
      "web_app": {
        "url": "https://kobravania.github.io/klyro_app/"
      }
    }
  }'
```

## Важно:

- Кнопка появится только после того, как пользователь начнёт диалог с ботом (отправит `/start`)
- Текст кнопки может быть на любом языке
- Кнопка будет видна внизу экрана чата с ботом

## После настройки:

Пользователи увидят кнопку внизу чата с ботом, нажав на которую откроется ваше Web App приложение Klyro!
